Incident Tracker — веб-приложение для регистрации и отслеживания инцидентов на предприятии. Сотрудники могут создавать инциденты, отслеживать их статус и оставлять комментарии. Администраторы имеют расширенные права управления.

Пользователи
- Сотрудник
  - Создавать новые инциденты
  - Редактировать свои инциденты
  - Просматривать детали инцидентов
- Администратор
  - Просматривать и редактировать все инциденты
  - Изменять статус и приоритет инцидентов
  - Оставлять комментарии к инцидентам

Инциденты
- Заголовок и описание
- Дата и время создания
- Статус: Новый, В работе, Закрыт
- Приоритет: Низкий, Средний, Высокий
- Комментарии к инцидентам

Фильтры и поиск
- Фильтрация по статусу, приоритету, дате создания
- Поиск по заголовку и описанию

Технологии
- Backend: PHP Yii2
- Frontend: HTML, CSS, JavaScript
- База данных: MySQL
